## :pushpin:풀이과정







풀이방법이 직관적으로 떠오르는 문제이다. 탐색하며 최고기울기를 저장해놓고 그 기울기보다 크면 보이는 빌딩으로 세면 된다.







## :pushpin:코드





```c++
#include <iostream>
#include <algorithm>
using namespace std;
int N, height[51], building[51];

void input(){
    cin >> N;
    for(int i = 0; i < N; i++){
        cin >> height[i];
    }
}

void solve(){
    for(int i = 0; i < N; i++){
        double MaxGradient = -1000000000;
        for(int j = i + 1; j < N; j++){
            double temp = (double)(height[j] - height[i]) / (j - i);
            if(temp <= MaxGradient){
                continue;
            }
            MaxGradient = temp;
            building[i]++;
            building[j]++;
        }
    }
    cout << *max_element(building, building + N) << "\n";
}

int main(){
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    input();
    solve();
    return 0;
}
```
